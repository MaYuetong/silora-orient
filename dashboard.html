Publish / Unpublish Workflow
To publish a series:
1. Put image in images/filename.jpg   2. Open content/collections.json   3. Set "published": true + add image filename   4. git push → live in 30 seconds
To unpublish (out of stock / pause): Set "published": false in content/collections.json → push. Series disappears from site immediately.
